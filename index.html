Easy setup guide

Get it up and running quickly

1

Install these first

  • Ruby and Rubygems
  • NodeJS
2

Get it on your system

  • Click here to download chopstick
  • Extract the chopstick folder
3

Setup chopstick

  • Open the chopstick folder in the terminal
  • a. Install bundler
    • gem update --system
    • gem install bundler
  • b. Install bower
    • npm install -g bower
    • bower install
  • c. Install gulp
    • npm install -g gulp
    • npm install
4

Enjoy Chopstick

  • Just run
  • gulp watch
